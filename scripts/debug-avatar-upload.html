🔧 头像上传调试页面

调试说明： 这个页面用于诊断头像上传问题，会显示详细的请求和响应信息。

使用 Fetch API 上传

// 推荐方式：使用 FormData const formData = new FormData(); formData.append('avatar', file); const response = await fetch('/api/admin/upload/avatar', { method: 'POST', headers: { 'Authorization': `Bearer ${token}` // 不要手动设置 Content-Type }, body: formData });

使用 Axios 上传

// 推荐方式：使用 FormData const formData = new FormData(); formData.append('avatar', file); const response = await axios.post('/api/admin/upload/avatar', formData, { headers: { 'Authorization': `Bearer ${token}` // 不要手动设置 Content-Type } });

使用 XMLHttpRequest 上传

// 传统方式：使用 FormData const formData = new FormData(); formData.append('avatar', file); const xhr = new XMLHttpRequest(); xhr.open('POST', '/api/admin/upload/avatar'); xhr.setRequestHeader('Authorization', `Bearer ${token}`); // 不要手动设置 Content-Type xhr.onload = function() { if (xhr.status === 200) { const response = JSON.parse(xhr.responseText); console.log('上传成功:', response); } }; xhr.send(formData);

使用传统 Form 提交

<form action="/api/admin/upload/avatar" method="post" enctype="multipart/form-data"> <input type="file" name="avatar" /> <input type="hidden" name="token" value="your_token" /> <button type="submit">上传</button> </form>

支持格式：JPG、PNG、GIF、WebP | 最大大小：2MB
